# Age of Sigmar Belarus Landing Page

Статический лендинг локального комьюнити Warhammer Age of Sigmar в Беларуси.
Размещён на GitHub Pages с кастомным доменом aos.by.

## Как обновлять сайт
- Новости, события и ссылки редактируются в `index.html`.
- Турниры добавляются в `tournaments.html` и отдельными папками в `tournaments/`.
- После пуша изменения автоматически деплоятся.
